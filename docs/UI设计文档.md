# UI设计文档

## 设计概述

基于UI目录中的页面设计图，本项目采用现代化的移动端优先设计，具有简洁清晰的界面风格和良好的用户体验。

## 设计规范

### 色彩规范
- **主色调**：蓝色系 (#1890ff)
- **辅助色**：绿色 (#52c41a)、橙色 (#fa8c16)、红色 (#f5222d)
- **中性色**：灰色系 (#f5f5f5, #d9d9d9, #595959, #262626)
- **背景色**：白色 (#ffffff)、浅灰 (#fafafa)

### 字体规范
- **主标题**：18px, 粗体
- **副标题**：16px, 中等
- **正文**：14px, 常规
- **辅助文字**：12px, 常规
- **小字**：10px, 常规

### 间距规范
- **页面边距**：16px
- **组件间距**：12px
- **内容间距**：8px
- **小间距**：4px

### 组件规范
- **按钮高度**：44px
- **输入框高度**：44px
- **卡片圆角**：8px
- **按钮圆角**：6px

## 响应式设计

### 移动端 (320px - 768px)
- 单列布局
- 大按钮设计
- 简化导航
- 触摸友好

### 平板端 (768px - 1024px)
- 双列布局
- 中等按钮
- 扩展导航
- 混合交互

### 桌面端 (1024px+)
- 多列布局
- 小按钮
- 完整导航
- 鼠标交互

## 交互设计

### 导航设计
- **底部导航**：首页、任务、收益、团队、我的
- **顶部导航**：返回、标题、操作
- **侧边导航**：功能菜单

### 状态反馈
- **加载状态**：骨架屏、加载动画
- **成功状态**：成功提示、状态更新
- **错误状态**：错误提示、重试机制
- **空状态**：空数据提示、引导操作

### 手势交互
- **滑动**：页面切换、列表滚动
- **点击**：按钮操作、链接跳转
- **长按**：菜单显示、快捷操作
- **下拉**：刷新数据、加载更多

## 无障碍设计

### 视觉无障碍
- 色彩对比度符合WCAG标准
- 字体大小可调节
- 图标配有文字说明
- 重要信息突出显示

### 操作无障碍
- 按钮大小符合触摸标准
- 操作反馈及时明确
- 键盘导航支持
- 语音提示支持

- **按钮**：主要按钮、次要按钮、文字按钮

### 布局组件
- **页面容器**：页面布局、内容区域
- **导航组件**：顶部导航、底部导航、侧边导航
