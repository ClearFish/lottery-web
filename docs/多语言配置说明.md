# 多语言配置说明

## 概述

本项目支持多语言功能，参考 `game-h5` 项目的语言代码设置。默认语言为越南语（`yuenan`）。

## 支持的语言

| 语言代码 | 语言名称 | 国旗图标 | 说明 |
|---------|---------|---------|------|
| `yuenan` | Tiếng Việt | 🇻🇳 | 越南语（默认） |
| `zh-cn` | 简体中文 | 🇨🇳 | 中文 |
| `en` | English | 🇺🇸 | 英语 |

## 文件结构

```
src/
├── locales/
│   ├── index.ts              # i18n 主配置
│   ├── locales.ts            # 语言文件导入
│   └── lang/

│       ├── zh-CN.json        # 中文翻译
│       └── en.json           # 英语翻译
├── utils/
│   └── lang.ts               # 语言工具函数
└── components/
    └── LanguageSwitcher/     # 语言切换组件
        └── index.vue
```

## 核心功能

### 1. 自动语言参数注入

所有API请求都会自动添加 `lang` 参数：

```typescript
// 请求拦截器自动添加
const currentLang = getCurrentLanguage();
config.params = { ...config.params, lang: currentLang };
```

### 2. 语言工具函数

```typescript
import { 
  getCurrentLanguage,    // 获取当前语言
  setLanguage,          // 设置语言
  getLanguageName,      // 获取语言名称
  getLanguageFlag,      // 获取语言国旗
  getLanguageDisplayText, // 获取完整显示文本
  getLanguageOptions     // 获取所有语言选项
} from '@/utils/lang'
```

### 3. 语言切换组件

```vue
<template>
  <LanguageSwitcher />
</template>

<script setup>
import LanguageSwitcher from '@/components/LanguageSwitcher/index.vue'
</script>
```

## 使用方法

### 1. 在组件中使用翻译

```vue
<template>
  <div>{{ $t('demo.tabbar.home') }}</div>
</template>

<script setup>
import { $t } from '@/locales'
</script>
```

### 2. 编程式语言切换

```typescript
import { setLocale } from '@/locales'
import { setLanguage } from '@/utils/lang'

// 切换语言
setLocale('yuenan')  // 设置i18n语言
setLanguage('yuenan') // 设置localStorage
```

### 3. 获取当前语言

```typescript
import { getCurrentLanguage } from '@/utils/lang'

const currentLang = getCurrentLanguage() // 'yuenan'
```

## 配置说明

### 1. 默认语言设置

```typescript
// src/locales/index.ts
const defaultLanguage = "yuenan"; // 默认越南语
```

### 2. 请求自动添加语言参数

```typescript
// src/utils/http/index.ts
// 所有请求自动添加 lang 参数
const currentLang = getCurrentLanguage();
config.params = { ...config.params, lang: currentLang };
```

### 3. 语言优先级

1. 请求参数中的 `lang`
2. localStorage 中保存的语言设置
3. 默认语言 `yuenan`

## 添加新语言

### 1. 创建语言文件

```json
// src/locales/lang/new-lang.json
{
  "demo": {
    "tabbar": {
      "home": "Home"
    }
  }
}
```

### 2. 更新类型定义

```typescript
// src/typings/i18n.d.ts
type LangType = "zh-cn" | "en" | "yuenan" | "new-lang";
```

### 3. 更新语言配置

```typescript
// src/locales/locales.ts
import newLangLocale from "./lang/new-lang.json";

const locales = {
  "zh-cn": zhCNLocale,
  en: enLocale,
  yuenan: yuenanLocale,
  "new-lang": newLangLocale
};
```

### 4. 更新语言工具

```typescript
// src/utils/lang.ts
export const SUPPORTED_LANGUAGES = {
  YUENAN: 'yuenan',
  ZH_CN: 'zh-cn',
  EN: 'en',
  NEW_LANG: 'new-lang'  // 添加新语言
} as const
```

## 注意事项

1. **语言代码一致性**: 确保所有地方使用的语言代码一致
2. **翻译完整性**: 新语言需要完整的翻译文件
3. **类型安全**: 更新TypeScript类型定义
4. **测试验证**: 确保语言切换功能正常工作

## 参考

- 参考 `game-h5` 项目的语言代码设置
- 语言代码使用 `yuenan` 而不是标准的 `vi` 或 `vi-VN`
- 默认语言为越南语，符合项目需求
