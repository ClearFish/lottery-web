## 快速开始

### 1. 环境要求
- Node.js >= 16.0.0
- MySQL >= 8.0
- Redis >= 6.0
- pnpm (推荐包管理器)

### 2. 项目启动
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

### 3. 数据库初始化
```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE task_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 导入表结构
mysql -u root -p task_platform < database/schema.sql

# 导入初始数据
mysql -u root -p task_platform < database/init.sql
```

## 技术特色

### 前端技术
- **Vue 3 + TypeScript**：现代化前端框架
- **Vite**：快速构建工具
- **Tailwind CSS**：原子化CSS框架
- **Pinia**：轻量级状态管理
- **响应式设计**：适配各种设备

### 后端技术
- **Node.js + Express**：高性能服务端
- **MySQL**：关系型数据库
- **Redis**：缓存和会话管理
- **JWT**：安全身份验证
- **RESTful API**：标准化接口设计

### 安全特性
- 密码加密存储
- JWT身份验证
