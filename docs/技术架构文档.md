# 技术架构文档

## 前端架构

### 技术栈选择
- **Vue 3**：采用 Composition API，提供更好的类型支持和性能
- **TypeScript**：类型安全，提高代码质量
- **Vite**：快速构建工具，支持热更新
- **Tailwind CSS**：原子化CSS框架，快速样式开发
- **Pinia**：Vue 3 推荐的状态管理库
- **Vue Router**：单页面应用路由管理

## 后端架构

### 技术栈建议
- **Node.js + Express**：服务端运行环境
- **MySQL**：关系型数据库
- **Redis**：缓存和会话存储
- **JWT**：身份验证
- **Multer**：文件上传处理
- **Nodemailer**：邮件发送

### 数据库设计

#### 核心表结构
```sql
-- 用户表
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  username VARCHAR(50) UNIQUE NOT NULL,
  phone VARCHAR(20) UNIQUE NOT NULL,
  password VARCHAR(255) NOT NULL,
  invite_code VARCHAR(20) UNIQUE NOT NULL,
  parent_id INT,
  balance DECIMAL(10,2) DEFAULT 0,
  status ENUM('active', 'inactive', 'banned') DEFAULT 'active',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (parent_id) REFERENCES users(id)
);

-- 任务表
CREATE TABLE tasks (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(200) NOT NULL,
  category ENUM('douyin', 'kuaishou', 'sms', 'xiaohongshu', 'toutiao') NOT NULL,
  reward DECIMAL(8,2) NOT NULL,
  total_count INT NOT NULL,
  remaining_count INT NOT NULL,
  keywords TEXT,
  description TEXT NOT NULL,
  requirements TEXT NOT NULL,
  video_materials JSON,
  image_materials JSON,
  status ENUM('active', 'inactive', 'completed') DEFAULT 'active',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 用户任务表
CREATE TABLE user_tasks (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  task_id INT NOT NULL,
  status ENUM('pending', 'in_progress', 'completed', 'abandoned') DEFAULT 'pending',
  completed_at TIMESTAMP NULL,
  reward_amount DECIMAL(8,2) DEFAULT 0,
  submission_data JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (task_id) REFERENCES tasks(id)
);

-- 提现记录表
CREATE TABLE withdrawals (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  amount DECIMAL(10,2) NOT NULL,
  method ENUM('alipay', 'wechat', 'bank') NOT NULL,
  account VARCHAR(100) NOT NULL,
  status ENUM('pending', 'approved', 'rejected', 'completed') DEFAULT 'pending',
  admin_notes TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  processed_at TIMESTAMP NULL,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 收益记录表
CREATE TABLE earnings (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  type ENUM('task', 'promotion', 'bonus') NOT NULL,
  amount DECIMAL(8,2) NOT NULL,
  source_id INT,
  description VARCHAR(200),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

## 安全考虑

### 前端安全
1. **输入验证**：所有用户输入进行客户端验证
2. **XSS防护**：使用Vue的内置转义机制
3. **CSRF防护**：使用JWT token验证
4. **敏感信息保护**：不在前端存储敏感信息

### 后端安全
1. **密码加密**：使用bcrypt加密存储
2. **JWT验证**：所有API请求验证JWT token
3. **SQL注入防护**：使用参数化查询
4. **文件上传安全**：限制文件类型和大小
5. **接口限流**：防止恶意请求

## 性能优化

### 前端优化
1. **代码分割**：按路由懒加载组件
2. **图片优化**：使用WebP格式，懒加载
3. **缓存策略**：合理使用浏览器缓存
4. **打包优化**：Tree shaking，压缩代码

### 后端优化
1. **数据库索引**：为常用查询字段建立索引
2. **Redis缓存**：缓存热点数据
3. **分页查询**：大数据量分页处理
4. **CDN加速**：静态资源使用CDN
