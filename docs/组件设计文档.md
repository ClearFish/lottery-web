# 组件设计文档

## 组件设计概述

基于UI设计图，本项目采用组件化开发模式，将页面拆分为可复用的组件，提高开发效率和代码维护性。

## 基础组件

### 1. 导航组件

#### NavBar 顶部导航栏
```vue
<template>
  <div class="navbar">
    <div class="navbar-left">
      <slot name="left">
        <button @click="goBack" v-if="showBack">
          <SvgIcon name="arrow-left" />
        </button>
      </slot>
    </div>
    <div class="navbar-center">
      <h1 class="navbar-title">{{ title }}</h1>
    </div>
    <div class="navbar-right">
      <slot name="right">
        <button @click="handleAction" v-if="actionText">
          {{ actionText }}
        </button>
      </slot>
    </div>
  </div>
</template>
```

**Props**:
- `title`: 页面标题
- `showBack`: 是否显示返回按钮
- `actionText`: 右侧操作按钮文字

#### Tabbar 底部导航栏
```vue
<template>
  <div class="tabbar">
    <div 
      v-for="item in tabs" 
      :key="item.name"
      class="tabbar-item"
      :class="{ active: currentTab === item.name }"
      @click="switchTab(item.name)"
    >
      <SvgIcon :name="item.icon" />
      <span class="tabbar-label">{{ item.label }}</span>
    </div>
  </div>
</template>
```

**功能**:
- 底部导航切换
- 图标和文字显示
- 当前页面高亮
- 路由跳转

### 2. 表单组件

#### Input 输入框
```vue
<template>
  <div class="input-wrapper">
    <label v-if="label" class="input-label">{{ label }}</label>
    <div class="input-container">
      <input
        v-model="modelValue"
        :type="type"
        :placeholder="placeholder"
        :disabled="disabled"
        class="input-field"
        @input="handleInput"
        @focus="handleFocus"
        @blur="handleBlur"
      />
      <button v-if="showClear && modelValue" @click="clear" class="input-clear">
        <SvgIcon name="close" />
      </button>
    </div>
    <div v-if="error" class="input-error">{{ error }}</div>
  </div>
</template>
```

**Props**:
- `modelValue`: 输入值
- `type`: 输入类型
- `label`: 标签文字
- `placeholder`: 占位符
- `error`: 错误信息
- `disabled`: 是否禁用

#### Button 按钮
```vue
<template>
  <button
    class="btn"
    :class="[
      `btn-${type}`,
      `btn-${size}`,
      { 'btn-disabled': disabled, 'btn-loading': loading }
    ]"
    :disabled="disabled || loading"
    @click="handleClick"
  >
    <SvgIcon v-if="loading" name="loading" class="btn-loading-icon" />
    <SvgIcon v-if="icon && !loading" :name="icon" />
    <span v-if="$slots.default"><slot /></span>
  </button>
</template>
```

**Props**:
- `type`: 按钮类型 (primary, secondary, danger)
- `size`: 按钮大小 (small, medium, large)
- `disabled`: 是否禁用
- `loading`: 加载状态
- `icon`: 图标名称

## 组件设计原则

### 1. 单一职责
每个组件只负责一个功能，保持组件的简洁和可维护性。

### 2. 可复用性
组件设计时考虑复用场景，通过props和slots提供灵活性。

### 3. 可配置性
通过props提供丰富的配置选项，满足不同使用场景。

### 4. 类型安全
使用TypeScript提供完整的类型定义，确保开发时的类型安全。

### 5. 性能优化
- 使用v-memo优化列表渲染
- 合理使用computed和watch
- 避免不必要的重新渲染

## 组件使用示例

### 页面中使用组件
```vue
<template>
  <div class="task-page">
    <NavBar title="任务中心" />
    
    <TaskFilter 
      :categories="taskCategories"
      @filter="handleFilter"
    />
    
    <div class="task-list">
      <TaskCard 
        v-for="task in tasks" 
        :key="task.id"
        :task="task"
        :task-status="getTaskStatus(task.id)"
        @claim="handleClaimTask"
        @click="goToTaskDetail(task.id)"
      />
    </div>
    
    <Tabbar :current-tab="'tasks'" />
  </div>
</template>
```

### 组件间通信
```vue
<script setup lang="ts">
// 父子组件通信
const emit = defineEmits<{
  claim: [taskId: number]
  click: [taskId: number]
}>()

// 全局状态管理
const userStore = useUserStore()
const taskStore = useTaskStore()

// 组件内部状态
const loading = ref(false)
const error = ref('')

const handleClaim = async (taskId: number) => {
  loading.value = true
  try {
    await taskStore.claimTask(taskId)
    emit('claim', taskId)
  } catch (err) {
    error.value = err.message
  } finally {
    loading.value = false
  }
}
</script>
```

## 组件测试

### 单元测试示例
```typescript
import { mount } from '@vue/test-utils'
import TaskCard from '@/components/TaskCard.vue'

describe('TaskCard', () => {
  it('renders task information correctly', () => {
    const task = {
      id: 1,
      name: '测试任务',
      reward: 10.00,
      description: '任务描述'
    }
    
    const wrapper = mount(TaskCard, {
      props: { task }
    })
    
    expect(wrapper.find('.task-title').text()).toBe('测试任务')
    expect(wrapper.find('.task-reward').text()).toBe('¥10.00')
  })
  
  it('emits claim event when claim button is clicked', async () => {
    const wrapper = mount(TaskCard, {
      props: { task: mockTask }
    })
    
    await wrapper.find('.claim-button').trigger('click')
    
    expect(wrapper.emitted('claim')).toBeTruthy()
    expect(wrapper.emitted('claim')[0]).toEqual([mockTask.id])
  })
})
```
