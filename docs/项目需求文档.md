# 任务分发平台 - 项目需求文档

## 项目概述

本项目是一个任务分发平台，用户可以通过完成各类任务获得收益，支持多平台任务分发和三级分销体系。

## 功能模块

### 1. 网站首页
**页面布局**：
- **顶部导航栏**：Logo、用户头像、消息通知
- **Banner轮播图**：展示平台特色和活动信息
- **推荐游戏区域**：热门游戏推广卡片
- **任务大厅**：
  - 任务分类网格布局（抖音、快手、短信、小红书、头条）
  - 分类图标和描述
  - 快速进入各分类任务
- **收益概览**：今日收益、总收益、提现按钮
- **底部导航栏**：首页、任务、收益、团队、我的

**交互功能**：
- 轮播图自动播放和手动切换
- 任务分类点击跳转
- 快速任务操作
- 消息通知提醒

### 2. 用户系统

#### 2.1 登录页面
**页面布局**：
- **页面标题**：登录
- **Logo区域**：平台Logo展示
- **登录表单**：
  - 手机号输入框
  - 密码输入框
  - 记住密码选项
  - 登录按钮
- **快速操作**：注册链接、忘记密码

**功能特性**：
- 手机号格式验证
- 密码强度提示
- 登录状态保持
- 错误信息提示

#### 2.2 注册页面
**页面布局**：
- **页面标题**：注册
- **注册表单**：
  - 手机号输入框
  - 短信验证码输入框
  - 密码设置输入框
  - 确认密码输入框
  - 邀请码输入框（可选）
  - 注册按钮
- **协议确认**：用户协议和隐私政策

**功能特性**：
- 手机号唯一性验证
- 短信验证码发送和验证
- 密码强度检查
- 邀请码绑定功能

### 3. 任务系统

#### 3.1 任务中心页面
**页面布局**：
- **顶部导航**：返回按钮、页面标题、筛选按钮
- **分类标签页**：抖音、快手、短信、小红书、头条
- **任务列表**：
  - 任务卡片布局
  - 任务标题和描述
  - 奖励金额显示
  - 剩余数量显示
  - 任务状态标识
- **筛选功能**：状态筛选、难度筛选、奖励筛选

**交互功能**：
- 分类切换
- 任务搜索
- 任务筛选
- 任务领取

#### 3.2 任务详情页面
**页面布局**：
- **任务头部**：任务标题、奖励金额、剩余数量
- **任务信息**：
  - 任务描述
  - 完成要求
  - 关键词标签
  - 预计时间
- **素材资源**：
  - 视频素材预览和下载
  - 图片素材预览和下载
  - 参考示例
- **操作按钮**：领取任务、收藏任务

**功能特性**：
- 素材预览功能
- 素材下载功能
- 任务收藏功能
- 任务分享功能

#### 3.3 任务提交页面
**页面布局**：
- **任务信息回顾**：任务名称、要求、奖励
- **提交表单**：
  - 完成截图上传
  - 提交说明输入
  - 相关链接输入
  - 备注信息
- **提交按钮**：确认提交

**功能特性**：
- 多文件上传
- 图片预览
- 表单验证
- 提交确认

### 4. 任务管理

#### 4.1 待完成任务列表
- 已领取但未完成的任务
- 任务进度跟踪
- 剩余时间提醒
- 任务操作按钮（开始/提交/放弃）

#### 4.2 已完成任务列表
- 历史完成任务记录
- 任务完成时间
- 获得奖励金额
- 任务评价

### 5. 推广系统

#### 5.1 推广海报生成
- 个性化海报模板
- 用户信息自动填充
- 二维码自动生成
- 分享链接生成

#### 5.2 邀请码系统
- 唯一邀请码生成
- 邀请码绑定
- 邀请关系建立

### 6. 个人中心

#### 6.1 个人中心页面
**页面布局**：
- **用户信息区域**：头像、用户名、手机号
- **收益概览**：余额、今日收益、总收益
- **功能菜单**：
  - 我的任务
  - 收益记录
  - 提现管理
  - 我的团队
  - 推广分享
  - 帮助手册
  - 设置

#### 6.2 提现页面
**页面布局**：
- **余额显示**：当前可用余额
- **提现金额输入**：金额输入框、全部提现按钮
- **提现方式选择**：支付宝、微信、银行卡
- **提现账户设置**：账户信息输入
- **提现说明**：提现规则和注意事项
- **提现按钮**：确认提现

**功能特性**：
- 最小/最大提现金额限制
- 提现方式验证
- 提现手续费计算
- 提现确认流程

#### 6.3 提现记录页面
**页面布局**：
- **筛选功能**：状态筛选、时间筛选
- **记录列表**：
  - 提现金额
  - 提现方式
  - 提现状态
  - 申请时间
  - 处理时间
- **状态标识**：待审核、已通过、已拒绝、已完成

#### 6.4 收益记录页面
**页面布局**：
- **收益统计**：总收益、今日收益、本月收益
- **收益类型筛选**：任务收益、推广收益、奖励收益
- **时间筛选**：今日、本周、本月、自定义
- **收益列表**：
  - 收益类型图标
  - 收益金额
  - 收益来源
  - 收益时间
  - 余额变化

#### 6.5 我的团队页面
**页面布局**：
- **团队统计**：总成员数、各级成员数、团队收益
- **成员列表**：
  - 成员头像和昵称
  - 加入时间
  - 贡献收益
  - 成员状态
- **层级结构**：一级、二级、三级成员展示

#### 6.6 推广分享页面
**页面布局**：
- **推广海报**：个性化海报展示
- **二维码**：邀请二维码
- **分享链接**：邀请链接复制
- **推广统计**：邀请人数、推广收益
- **分享按钮**：微信、QQ、朋友圈分享

#### 6.7 帮助手册页面
**页面布局**：
- **搜索功能**：问题搜索
- **分类导航**：注册登录、任务相关、提现相关、推广相关
- **常见问题**：问题列表和答案
- **联系客服**：在线客服入口
- **意见反馈**：反馈提交

#### 6.8 消息通知页面
**页面布局**：
- **消息分类**：系统通知、任务通知、收益通知
- **消息列表**：
  - 消息标题
  - 消息内容
  - 发送时间

- **操作功能**：标记已读、删除消息、全部已读

## 技术架构

### 前端技术栈
- Vue 3 + TypeScript
- Vite 构建工具
- Tailwind CSS 样式框架
- Vue Router 路由管理
- Pinia 状态管理
- Axios HTTP请求

### 后端技术栈（建议）
- Node.js + Express/Koa
- MySQL/PostgreSQL 数据库
- Redis 缓存
- JWT 身份验证
- 文件上传服务

### 数据库设计

#### 用户表 (users)
```sql
- id: 主键
- username: 用户名
- phone: 手机号
- password: 密码
- invite_code: 邀请码
- parent_id: 上级ID
- balance: 账户余额
- created_at: 创建时间
- updated_at: 更新时间
```

#### 任务表 (tasks)
```sql
- id: 主键
- name: 任务名称
- category: 任务分类
- reward: 奖励金额
- total_count: 总数量
- remaining_count: 剩余数量
- keywords: 关键词
- description: 任务描述
- video_materials: 视频素材
- image_materials: 图片素材
- status: 任务状态
- created_at: 创建时间
```

#### 用户任务表 (user_tasks)
```sql
- id: 主键
- user_id: 用户ID
- task_id: 任务ID
- status: 任务状态（待完成/已完成/已放弃）
- completed_at: 完成时间
- reward_amount: 获得奖励
- created_at: 领取时间
```

#### 提现记录表 (withdrawals)
```sql
- id: 主键
- user_id: 用户ID
- amount: 提现金额
- method: 提现方式
- account: 提现账户
- status: 提现状态
- created_at: 申请时间
- processed_at: 处理时间
```

## 开发计划

### 第一阶段：基础功能
1. 用户注册登录系统
2. 任务列表和详情页面
3. 基础的任务领取和完成流程

### 第二阶段：核心功能
1. 个人中心开发
2. 提现功能实现
3. 收益统计系统

### 第三阶段：高级功能
1. 推广系统开发
2. 三级分销功能
3. 客服系统集成

### 第四阶段：优化完善
1. 性能优化
2. 安全加固
3. 用户体验优化

## 注意事项

1. **安全性**：用户密码加密存储，API接口鉴权
2. **性能**：数据库索引优化，缓存策略
3. **用户体验**：响应式设计，加载状态提示
4. **合规性**：任务内容审核，用户协议完善
5. **扩展性**：模块化设计，便于后续功能扩展
